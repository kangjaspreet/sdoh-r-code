# R Code For SDOH Data from *Correlating Social Determinants of Health with Causes of Premature Death in California 2015-2024*

R code for extracting and processing the American Community Survey SDOH tract-level data from the manuscript "*Correlating Social Determinants of Health with Causes of Premature Death in California 2015-2024*". The code is generalizable such that the SDOH data can be pulled for all census tracts in any U.S. State.
